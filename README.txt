deploy/build package

This package contains tools that are used for building and deploying
the LSST software stack.  This package includes the following
contents:

  README.txt           this file
  buildtemplates       templates for creating a compliant LSST package
  deploy               special LSST installable packages and tools
                         employed by users as part of the stack
                         deployment process.
  servertools          tools used to set up the software server
  usertools            tools which can be executed directly by users
                         wil making a package LSST compliant.
